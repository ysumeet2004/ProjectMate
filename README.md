# ProjectMate - Collaborate Like a Pro

**Discover & collaborate on projects with creators worldwide**

ProjectMate connects developers, designers, product managers, and entrepreneurs to find teammates, launch side projects, and build amazing things together. Search for open projects, find your next collaborators, and join teams building the future.

## 🎯 The Problem

- You have a great project idea but need a co-founder or team member
- You're looking for your next side hustle or full-time opportunity
- Finding trustworthy collaborators is hard and time-consuming
- Existing platforms aren't built for project collaboration

## ✨ The Solution

ProjectMate is a **collaboration platform** where:

✅ Creators post open projects looking for teammates  
✅ Developers & designers find projects that match their skills  
✅ Teams are built based on verified profiles & past work  
✅ Collaboration happens in real-time within the platform  
✅ Success is tracked and celebrated

## 🚀 Key Features

- **Project Discovery**: Browse by domain, skills, commitment level, and timeline
- **Smart Matching**: Find projects aligned with your expertise and interests
- **Verified Profiles**: See creator history, completed projects, and user reviews
- **Built-in Collaboration**: Chat, file sharing, and project management tools
- **Success Tracking**: Portfolio, testimonials, and collaboration history
- **Real-time Notifications**: Never miss opportunities that match you

## 🛠 Tech Stack

- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Frontend**: EJS templates + Bootstrap + Custom CSS
- **Security**: JWT, CSRF protection, bcrypt, Helmet.js
- **Deployment**: Ready for Render/Heroku

## 📦 Getting Started

### Prerequisites

- Node.js 16+
- MongoDB
- npm

### Installation

```bash
git clone https://github.com/ysumeet2004/ProjectMate.git
cd ProjectMate
npm install
```

### Configuration

Create a `.env` file in the root directory:

```env
PORT=8000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
NODE_ENV=development
```

### Running Locally

```bash
npm run devStart
```

Visit `http://localhost:8000`
